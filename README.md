# 📚 Student Management System in C++

A console-based application built using C++ to manage student records efficiently through a menu-driven system.

## 🚀 Features

* Add new students
* Search students by ID
* Update student information
* Delete student records
* Display all students
* Calculate average grades
* Find the highest-grade student
* Save and load data using file handling

## 🛠️ Technologies & Concepts Used

* C++
* Object-Oriented Programming (OOP)
* STL Vectors
* File Handling
* Functions & Modular Design
* Problem Solving

## 💡 Project Purpose

This project was built to strengthen my understanding of Object-Oriented Programming, data management, and file handling while practicing clean logic implementation and problem-solving skills.

## ▶️ How to Run

```bash
g++ main.cpp -o app
./app
```
