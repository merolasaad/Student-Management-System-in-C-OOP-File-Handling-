# 📚 Student Management System in C++

A simple console-based application to manage student records using C++.

---

## 🚀 Features
- Add students
- Search by ID
- Delete student
- Update student
- Display all students
- Calculate average grade
- Find highest grade student
- File handling (save/load data)

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Vectors (STL)
- File Handling
- Functions & Menu System
- Problem Solving

---

## 💡 How to Run
Compile and run:
g++ main.cpp -o app
./app
